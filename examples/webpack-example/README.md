# What is webpack?

Webpack is a bundler, a bundler is a tool that is used to take multiple separate source files and combine them into one.
Webpack also enables preprocessing through the use of loaders like babel and external features like plugins.

# Why webpack?

Webpack enables use to design a build system specific to a web applications needs. We can use require statements to include other source files rather than manually adding `<script>` tags and referencing our sources. This removes the need to manage our order of which our dependencies/sources are loaded and enables far easier maintainable code.
